Social Buttons

Code

To add social icon use following markup:

<a href="#" class="social-btn sb-style-number sb-brandName">
  <i class="socicon-brandName"></i>
</a>

Examples with tooltips enabled:

  • HTML
  • Pug
<!-- Social Icon (Style 1) -->
<a class="social-btn sb-style-1 sb-facebook" href="#" data-toggle="tooltip" data-placement="top" title="Facebook">
  <i class="socicon-facebook"></i>
</a>

<!-- Social Icon Light Skin (Style 1) -->
<a class="social-btn sb-style-1 sb-facebook sb-light-skin" href="#" data-toggle="tooltip" data-placement="top" title="Facebook">
  <i class="socicon-facebook"></i>
</a>
//- Social Icon (Style 1)
