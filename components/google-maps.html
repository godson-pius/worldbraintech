Google Maps

Code

To be able to use Google Maps on your website you have to register Maps Javascript API key (Make sure to enable Geocoding API as well).
Please follow the link below for the instructions.

https://developers.google.com/maps/documentation/javascript/get-api-key
  • HTML
  • Pug
<!-- Default Map -->
<div class="google-map" data-height="270" data-address="New York, USA" data-zoom="9" data-disable-controls="true" data-scrollwheel="true" data-marker-title="We are here!"></div>
//- Default Map
.google-map(
  data-height = "270",
  data-address = "New York, USA",
  data-zoom = "9",
  data-disable-controls = "true",
  data-scrollwheel = "true",
  data-marker-title = "We are here!"
)

Code

To be able to use Google Maps on your website you have to register Maps Javascript API key (Make sure to enable Geocoding API as well).
Please follow the link below for the instructions.

https://developers.google.com/maps/documentation/javascript/get-api-key

To apply custom styles to the map pass styles in JSON format to the data attribute: data-styles. The best collection of Google Maps styles can be found at https://snazzymaps.com/

  • HTML
  • Pug
<!-- Custom Styling -->
<div class="google-map" data-height="270" data-address="New York, USA" data-zoom="9" data-disable-controls="true" data-scrollwheel="false" data-marker="path-to-map-marker-image" data-marker-title="We are here!" data-styles='[{ "styles in JSON format" }]'></div>
//- Custom Styling
.google-map(
  data-height = "270",
  data-address = "New York, USA",
  data-zoom = "9",
  data-disable-controls = "true",
  data-scrollwheel = "false",
  data-marker = "path-to-map-marker-image",
  data-marker-title = "We are here!",
  data-styles = '[{ "styles in JSON format" }]'
)
